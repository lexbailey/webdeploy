# WebDeploy

### Deploy files to the web via FTP

## Installation

To install from the github repo:

 - clone the repo
 - enter the repo directory
 - run `make`
 - run `make install`

you can now run `webdeploy`

## Usage

For detailed usage, see the man page.
(once installed type `man webdeploy`)
